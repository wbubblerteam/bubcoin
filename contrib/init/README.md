Sample configuration files for:
```
SystemD: bubcoind.service
Upstart: bubcoind.conf
OpenRC:  bubcoind.openrc
         bubcoind.openrcconf
CentOS:  bubcoind.init
macOS:   org.bubcoin.bubcoind.plist
```
